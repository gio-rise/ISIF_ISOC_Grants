# ISOC SPI 2026 Grant Program — Research

**Date:** 2026-04-16
**Context:** Rise is applying for the ISOC Foundation's Sustainable Peering Infrastructure (SPI) grant, Peering Ecosystem Support track (up to USD 25K). This research captures program requirements, deadlines, evaluation criteria, and strategic considerations.

## Findings

### Funding Tracks

| Track | Max Grant | Purpose |
|---|---|---|
| IXP Support | USD 50,000 | Establish new or enhance existing IXPs |
| Peering Ecosystem Support | USD 25,000 | Capacity building, training, community building for peering ecosystem |

### Timeline

| Milestone | Date |
|---|---|
| Application window opens | 3 March 2026 |
| Info session webinar | 10 March 2026, 8:00 AM EST |
| **Application deadline** | **23 April 2026, 21:00 UTC** (5:00 AM PHT on 24 April) |
| Fluxx registration deadline | 22 April 2026 (24 hours before deadline) |
| Review period | Up to 14 weeks (~late July / early August 2026) |
| Project duration | Up to 12 months |
| Reporting | Every ~6 months + final report 1 month after completion |
| Possible second window | Later in 2026 if funding remains |

### Eligibility

- Must be a legally registered organization
- Must have a bank account that can receive funds from a US-based foundation
- Must demonstrate successful experience managing similar work
- **IXP Support:** IXP must be "publicly beneficial, neutral, and not-for-profit"
- **Peering Ecosystem Support:** The not-for-profit restriction does not explicitly apply
- **For-profit caution:** Rise should frame around community/public-interest dimension
- **Geographic preference:** Strong preference for UN-recognized LDCs and SIDS — Philippines is neither
- Applications from non-LDC/non-SIDS countries explicitly welcome but without geographic preference

### Required Documents (Peering Ecosystem Support)

- Completed budget template
- Completed workplan template
- Minimum one document demonstrating community interest (letters of support, survey results, prior workshop attendance)

### Application Process

1. Register organization in Fluxx: https://isocfoundation.fluxx.io/apply/orgregistration
2. Log in: https://isocfoundation.fluxx.io/user_sessions/new
3. Select SPI program → Peering Ecosystem Support track
4. Complete application form within Fluxx
5. Upload required documents
6. Submit before 21:00 UTC on 23 April 2026

### Evaluation Criteria

Reviewed by Internet Society + ISOC Foundation + external experts:
1. **Project alignment** — supports overall SPI program objectives
2. **Applicant capacity** — knowledge, relationships, and experience to accomplish goals
3. **Project planning quality** — goals, outcomes, milestones, workplan clearly defined and aligned
4. **Geographic priority** — preference for LDCs and SIDS (works against Philippines)

No formal scoring rubric with point values published — criteria are qualitative.

### Program History

- Evolved from Internet Society's "Beyond the Net" grants program
- SPI is the dedicated vehicle for peering/IXP grants
- 2026 round reflects Foundation's 2030 Strategy with LDC/SIDS emphasis
- Beyond the Net archive: https://www.isocfoundation.org/grant-programme/beyond-the-net-grant-program/

### Fiscal Sponsor Option

If needed, organizations can use a fiscal sponsor. Both sponsor and sponsored org must register independently in Fluxx. Guidance: https://www.isocfoundation.org/wp-content/uploads/2021/12/Guidance-for-Applicants-Fiscal-Sponsors-2021-11-19-1.pdf
